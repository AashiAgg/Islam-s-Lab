import pandas as pd
import matplotlib.pyplot as plt

# Load the penguins.csv file into a DataFrame
df = pd.read_csv("penguins.csv")

# Create separate DataFrames for each species
adelie_data = df[df['species'] == 'Adelie']
chinstrap_data = df[df['species'] == 'Chinstrap']
gentoo_data = df[df['species'] == 'Gentoo']

# Create a scatterplot with all three species
plt.figure(figsize=(8, 6))

# Scatterplot for Adelie penguins
plt.scatter(adelie_data['bill_length_mm'], adelie_data['bill_depth_mm'], c='pink', label='Adelie', marker='o', s=100)

# Scatterplot for Chinstrap penguins
plt.scatter(chinstrap_data['bill_length_mm'], chinstrap_data['bill_depth_mm'], c='green', label='Chinstrap', marker='s', s=100)

# Scatterplot for Gentoo penguins
plt.scatter(gentoo_data['bill_length_mm'], gentoo_data['bill_depth_mm'], c='purple', label='Gentoo', marker='^', s=100)

plt.xlabel('Bill Length (mm)')
plt.ylabel('Bill Depth (mm)')
plt.title('Penguin Bill Length vs. Bill Depth')
plt.legend()

plt.show()

What do you observe from the graph? (2 marks)

We can observe that Adelie species has greater bill depth and small bill length where as Gentoo has longer bill length and smaller bill depth. Chinstrap has moderate depth and length.

Why did you choose this type of graph? (2 marks)

Because it clearly differentiates between the three species. Different colours data points defining each species makes it easier to understand.

import pandas as pd
import matplotlib.pyplot as plt


df = pd.read_csv("penguins.csv")

df['BMI'] = df['body_mass_g'] / ((df['flipper_length_mm'] / 1000) ** 2)  


mean_bmi_by_sex = df.groupby('sex')['BMI'].mean()


plt.figure(figsize=(8, 6))
mean_bmi_by_sex.plot(kind='bar', color=['green', 'purple'], alpha=0.7)
plt.xlabel('Sex')
plt.ylabel('Mean BMI')
plt.title('Mean BMI Penguins based on gender')
plt.xticks(rotation=0)
plt.show()

The bmi of female is smaller than male penguins. there isnt much difference but visible.
